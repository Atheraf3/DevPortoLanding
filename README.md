# DevPortoLanding

DevPortoLanding adalah sebuah aplikasi web berbasis React yang berfungsi sebagai landing page portofolio profesional bagi pengembang perangkat lunak. Proyek ini mengedepankan desain minimalis, performa tinggi, dan pengalaman pengguna yang responsif di berbagai perangkat.

## Fitur Utama

Aplikasi ini mencakup berbagai modul fungsional yang dirancang untuk kebutuhan profesional:

* **Navigasi Adaptif**: Sistem navigasi yang menyesuaikan tampilan berdasarkan posisi scroll dan tersedia dalam format menu dropdown untuk perangkat seluler.
* **Presentasi Visual**: Implementasi animasi transisi menggunakan Framer Motion untuk meningkatkan estetika interaksi tanpa mengorbankan performa.
* **Kategorisasi Keahlian**: Pengelompokan kompetensi teknis yang terorganisir ke dalam kategori Pengembangan Frontend, Backend, dan Manajemen Alat (Tools).
* **Galeri Proyek Terintegrasi**: Komponen showcase yang menampilkan detail proyek, tumpukan teknologi (tech stack), serta tautan langsung ke repositori kode dan demonstrasi aplikasi.
* **Integrasi Kontak**: Hubungan langsung ke saluran komunikasi profesional termasuk Email, WhatsApp, dan LinkedIn.

## Spesifikasi Teknologi

Pengembangan proyek ini menggunakan tumpukan teknologi modern berikut:

* **Library Utama**: React.js
* **Framework CSS**: Tailwind CSS
* **Mesin Animasi**: Framer Motion
* **Koleksi Ikon**: Lucide React
